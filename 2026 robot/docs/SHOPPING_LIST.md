## Where to Buy

### Online Stores (Reliable):
- **Amazon** - Fast shipping, good returns
- **AliExpress** - Cheapest prices, slower shipping (2-4 weeks)
- **eBay** - Good deals, varies by seller
- **Adafruit** - High quality, educational support
- **SparkFun** - Quality components, tutorials
- **RobotShop** - Specialized robot parts

### Physical Stores:
- **MicroCenter** (US) - Good selection, competitive prices
- **Fry's Electronics** (if still in your area)
- **Best Buy** - Limited selection but available
- **Local electronics hobby shops** - Support local!

## Money-Saving Tips

1. **Buy a Complete Kit** - Often cheaper than individual parts
2. **AliExpress/Banggood** - Save 30-50% if you can wait 3-4 weeks
3. **Bundle Deals** - Search for "Arduino starter kit" or "ESP32 robot kit"
4. **Used Parts** - Check eBay for used Arduino/ESP32 boards
5. **Start Minimal** - Get basic parts first, add sensors later

## Recommended First Purchase (Budget: ~$50)

For absolute beginners, I recommend:

```
✓ ESP32 Robot Car Kit from AliExpress ($40-50)
  Includes: ESP32, chassis, motors, motor driver, battery holder
  
This gives you everything to start!
```

## Next Steps After Purchase

1. Wait for parts to arrive
2. Download Arduino IDE
3. Install ESP32 or Arduino board support
4. Upload the provided code to your board
5. Build the Android app (already created for you)
6. Test and enjoy!

## Need Help Choosing?

**Complete Beginner?** → Buy an ESP32 Robot Car Complete Kit
**Have Some Experience?** → Buy individual parts (Option 1)
**Want Best Performance?** → ESP32 with 4WD chassis
**On Tight Budget?** → Arduino Nano + HC-06 + 2WD chassis (< $40 from AliExpress)
